Projet Annuaire Symfony

Contexte du Projet

Ce projet a été réalisé en trinôme dans le cadre du TD de Symfony. L'objectif était de développer une application d'annuaire en ligne permettant aux utilisateurs de créer, modifier, et gérer leur profil. Le projet visait à mettre en pratique les compétences acquises dans le cadre du cours et des TDs Symfony, tout en respectant des contraintes techniques et fonctionnelles.

Délai : Jusqu'au 28 septembre 2024.

Méthodes de Travail

Le projet a suivi une approche collaborative avec répartition des tâches entre les membres du trinôme. Nous avons utilisé :

  • GitHub pour le versionnement et le suivi du projet avec GitHub Projects.
  • Docker pour l'isolation des services et la portabilité de l'environnement.
  • Méthodes de projet agiles pour organiser et planifier les livrables.

Résultats Obtenus

  • Page principale affichant tous les profils visibles.
  • Fonctionnalité d'inscription avec vérification asynchrone de l'unicité du code profil.
  • Possibilité pour les utilisateurs d'éditer et de supprimer leur profil.
  • Système de gestion des rôles et accès administrateurs.
  • Mise en place d'une route pour afficher les données utilisateur au format JSON.
  • Mode maintenance configurable via le fichier services.yaml.
  • Mise en place de fixtures pour peupler la base de données
  • Mise en place d'une liste de favoris pour permettre de retrouver plus rapidement des proches
