Les Aventuriers du Graphe – Partie 2

Contexte du Projet

Ce projet fait suite à la Phase 1 et s'inscrit dans le cadre des Situations d'Apprentissage et d'Évaluation (SAE) du BUT Informatique. L'objectif de cette phase était de modéliser le plateau du jeu Les Aventuriers du Rail sous forme de graphe, pour implémenter des algorithmes avancés permettant d'optimiser les stratégies de jeu.

Les objectifs principaux comprenaient :

  • Représenter les routes du plateau sous forme de graphes.
  • Implémenter des algorithmes de parcours, comme Dijkstra, pour déterminer les chemins optimaux.
  • Analyser et valider les destinations réalisables pour chaque joueur.

Méthodes de Travail et Résultats

Nous avons appliqué une méthodologie agile, avec des itérations hebdomadaires pour valider chaque fonctionnalité. Les outils utilisés incluent :

  • GitHub pour la gestion de version.
  • JUnit pour les tests unitaires.
  • IntelliJ IDEA comme IDE principal.

Résultats obtenus

  • Modélisation réussie du plateau en tant que graphe.
  • Implémentation d'algorithmes de parcours pour calculer les chemins optimaux.
  • Interface graphique mise à jour pour afficher les routes en surbrillance lors des simulations.
